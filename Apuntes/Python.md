Compartir carpeta en red

    Seleccionamos la carpeta que queremos compartir con click secundario y abrimos con el terminal

    Escribimos "python" para ver si esta instalado en el ordenador, y sino lo instalamos

    Escribimos "python -m http.server -b 0.0.0.0

    ---

    Para acceder desde otro equipo debe estar conectado el servidor, ambos deben estar conectado a la misma red

    En el cliente debe abrir el navegador y poner la IP del otro equipo, dos puntos, 8000 sin espacios

    Ejemplo: 192.168.1.1:8000

    Nota: Se cierra la carpeta cuando se cierra el terminal
